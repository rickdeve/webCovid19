
# WebCovid19

Este proyecto consiste en obtener las estadisticas del virus COVI19 por cada pais y a nivel mundial, mediante una API  publicada en el siguien repositorio [NovelCovid](https://github.com/NovelCOVID/API).

## Creado en
<ul>
  <li>Angular 8</li>
  li>Angular 8</li>
  li>TypeScript</li>
</ul>
